## 簡介
使用 Express Handlebars 打造縮短網址產生器

### 功能
- 將輸入的原始網址轉換成短網址
- 點擊 `copy` 按鈕複製短網址
- 使用短網址連向原始網址的網站

## 開始使用
1. 進入URL_shortener
2. 輸入
```zsh
npm install
```
3. 開啟瀏覽器進入以下網址:
```zsh
http://localhost:3000
```
4. 若欲暫停使用
```zsh
ctrl + c
```

## 開發工具
- body-parser: 1.20.0
- express: 4.18.1
- express-handlebars: 4.0.2
- mongoose: 6.3.6
